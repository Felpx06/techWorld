Java: Programação Orientada a Objetos

Entenda o que faz do Java tão presente em sistemas corporativos e mobile.

História do Java

Em 1991, James Gosling iniciou o projeto “Oak” na Sun Microsystems, visando uma linguagem portátil para dispositivos como TVs. Lançado como Java em 1995, ganhou o lema “write once, run anywhere” devido à JVM.

Java evoluiu com releases regulares (Java 5 trouxe generics em 2004; Java 8 trouxe lambdas em 2014). É essencial em servidores, Android, big data, IoT e sistemas financeiros. Em 2010, a Oracle assumiu a plataforma.

Por que aprender Java?

  • ✔ Portabilidade via JVM em diversas plataformas
  • ✔ Estrutura robusta para grandes aplicações corporativas
  • ✔ Ecossistema maduro com frameworks como Spring e Hibernate
  • ✔ Base da programação Android até meados de 2019

Conceitos centrais

  • JVM: máquina virtual que interpreta bytecode Java
  • POO: princípios como encapsulamento, herança, polimorfismo
  • Tipos: Java é fortemente tipada — exige declaração de variáveis
  • Collections: List, Set, Map e streams possibilitam manipulação de dados poderosa

Exemplos Avançados

1. Classe com método e encapsulamento

Define uma classe `Pessoa` com atributos privados e método público.

public class Pessoa {
  private String nome;
  private int idade;

  public Pessoa(String nome, int idade) {
    this.nome = nome;
    this.idade = idade;
  }

  public void apresentar() {
    System.out.println("Olá, sou " + nome + ", tenho " + idade + " anos.");
  }
}

2. Uso de Collection e Stream

Filtra e processa os nomes com Stream API.

import java.util.*;
import java.util.stream.*;

List nomes = Arrays.asList("Ana","Bruno","Carlos","Daniela");
List filtrados = nomes.stream()
  .filter(n -> n.startsWith("C"))
  .map(String::toUpperCase)
  .collect(Collectors.toList());
System.out.println(filtrados);

3. Tratamento de exceções

Exemplo de `try-catch` para lidar com erros.

try {
  int div = 10 / 0;
} catch (ArithmeticException e) {
  System.out.println("Erro: divisão por zero!");
}

Conclusão

Java continua relevante graças à sua portabilidade, robustez e maturidade. Aprendê-lo é aprender uma linguagem voltada a arquitetura de software, sistemas corporativos e desenvolvimento moderno.
