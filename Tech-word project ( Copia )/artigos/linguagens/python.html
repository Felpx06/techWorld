Introdução ao Python

Simples. Poderoso. Ideal para quem está começando — e para quem já domina o mundo da programação.

O que é Python?

Python é uma linguagem de programação moderna, fácil de aprender e extremamente poderosa. Seu código é limpo, direto e muito próximo da linguagem humana — o que a torna ideal para iniciantes e ao mesmo tempo poderosa o suficiente para aplicações avançadas.

Quem criou o Python?

Python foi criado por Guido van Rossum no final da década de 1980. A primeira versão foi lançada em 1991 com o objetivo de ser uma linguagem simples, intuitiva e poderosa.

Desde então, Python se tornou uma das linguagens mais populares do mundo, sendo usada em inteligência artificial, análise de dados, automação, web, games, robótica e muito mais.

Exemplos de Python na prática

1. Exibir uma mensagem

Função `print()`: usada para exibir mensagens na tela.

print("Olá, mundo!")

2. Variáveis e tipos

Python detecta automaticamente o tipo da variável com base no valor atribuído.

nome = "Joana" idade = 30 altura = 1.68 print(nome, idade, altura)

3. Condições (if / else)

Use `if` para verificar condições. O `else` executa o código alternativo.

nota = 7 if nota >= 6: print("Aprovado") else: print("Reprovado")

4. Laço de repetição (for)

Use `for` para repetir algo várias vezes.

for i in range(1, 6): print(f"Passo {i}")

5. Listas e iteração

Listas armazenam coleções de dados, como vários nomes.

nomes = ["Ana", "Carlos", "Rafa"] for nome in nomes: print("Olá", nome)

6. Criando funções

Funções são blocos de código reutilizáveis.

def saudacao(nome): print(f"Bem-vindo, {nome}!") saudacao("Lucas")

7. Dicionários

Dicionários armazenam pares chave-valor.

pessoa = {"nome": "Marcos", "idade": 22} print(pessoa["nome"])

8. Orientação a Objetos (POO)

Classes representam modelos de objetos no mundo real.

class Pessoa: def __init__(self, nome): self.nome = nome def falar(self): print("Olá, meu nome é", self.nome) p = Pessoa("Gabriel") p.falar()

Conclusão

Python não é apenas uma linguagem para aprender: é uma ferramenta poderosa usada por empresas, universidades e governos. Simples para começar, forte para crescer. Agora que você viu os conceitos básicos, está pronto para escrever seus próprios programas e resolver problemas reais com código.
